# 📊 Student Performance Analysis

## 📌 Internship Task 1 – Data Analysis with Python

This project is part of my **Data Science & Data Analysis with Python Internship**. The objective of this task was to perform **Exploratory Data Analysis (EDA)** on the Student Performance dataset using Python and derive meaningful insights through data visualization.

---

## 🎯 Objective

To analyze student performance data, identify trends, and understand the relationship between different factors affecting students' academic results using Python.

---

## 📂 Dataset

- **Dataset:** Student Performance Dataset
- **Records:** 395
- **Features:** 33
- **Format:** CSV

---

## 🛠️ Tools & Technologies

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📋 Tasks Performed

- Imported and loaded the dataset.
- Explored the dataset structure and features.
- Checked data types and statistical summary.
- Verified missing values and duplicate records.
- Calculated the average final grade.
- Identified students scoring above 15 marks.
- Analyzed the relationship between study time and final grades.
- Compared academic performance based on gender.
- Created visualizations to better understand the dataset.
- Summarized the findings with meaningful conclusions.

---

## 📊 Visualizations Created

- 📈 Histogram of Final Grades
- 📉 Scatter Plot (Study Time vs Final Grade)
- 📊 Bar Chart (Average Grade by Gender)
- 🔥 Correlation Heatmap

---

## 🔍 Key Findings

- The dataset contains **395 student records** with **33 attributes**.
- No missing values or duplicate records were found.
- Study time shows a positive relationship with final grades.
- Most students scored in the medium grade range.
- Male and female students showed comparable academic performance.
- Data visualization helped identify important trends and patterns.
